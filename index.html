Mi interés por la programación y la resolución de problemas tecnológicos me ha permitido desarrollar una sólida base técnica, que sigo perfeccionando a través de la experiencia práctica y el aprendizaje continuo. Proyectos

Proyecto 1: Aplicación de Tareas

Desarrollé una aplicación para la gestión de tareas pendientes utilizando HTML, CSS y JavaScript. El proyecto permite a los usuarios agregar, editar y eliminar tareas, manteniendo un almacenamiento local para persistir los datos.

Creé un conversor de monedas en tiempo real utilizando la API de ExchangeRate, permitiendo a los usuarios convertir diferentes monedas con tasas de cambio actualizadas automáticamente.
